> \# Problema\
> Muitos usuários desistem de controlar seus gastos porque os apps atuais exigem muita entrada manual e oferecem pouca personalização. Este MVP deve resolver isso com uma experiência de conversa fluida e recomendações automáticas de economia.
> 
> \# Público-Alvo\
> Pessoas que querem começar a organizar suas finanças de forma prática e sem complicação, principalmente iniciantes.
> 
> \# Funcionalidades-Chave\
> 1. Registrar gastos via chat em linguagem natural.
> 2. Classificar automaticamente as transações.
